# 學員列表

進入「學員管理」頁面，可查看所屬精舍的所有學員。

## 功能說明

### 搜尋學員
在搜尋框中輸入學員編號、姓名或法名，即時篩選學員。

### 新增學員
1. 點擊「新增學員」按鈕
2. 填寫學員資料（學員編號、姓名、法名等）
3. 點擊確認儲存

### 編輯學員
點擊學員列表中的「編輯」按鈕，修改學員資料後儲存。

::: warning 注意
學員編號在系統中為唯一值，無法重複。
:::

### 匯入學員
點擊「匯入檔案」進入批次匯入頁面。詳細流程請見 [匯入學員](/student-import)。

### 匯出學員

點擊「匯出全部 (N)」按鈕可將學員資料下載為 Excel 檔。若目前有使用搜尋或精舍篩選，按鈕會切換成「匯出篩選結果 (N)」，僅匯出符合條件的學員。

匯出的欄位與[匯入範本](/student-import)完全一致，因此可以：
- 以 Excel 批次編輯姓名／法名後，再透過匯入功能一次更新；
- 作為精舍內學員資料的備份或稽核用。

::: tip 關於學員狀態
匯出會包含所有學員（含已停用），因為匯入時不會改變學員狀態，不會影響現有設定。
:::

## CT 管理員

CT 管理員可透過精舍篩選下拉選單，查看各精舍的學員資料。「匯出全部 / 匯出篩選結果」會呼叫後端整批取得所有符合條件的學員，匯出檔案會額外帶有「精舍名稱」欄位，與 CT 匯入範本完全對應。
