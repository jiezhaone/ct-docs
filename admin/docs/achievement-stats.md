# 達標情況統計

達標情況統計表顯示每位學員在八大修行項目的進度與達標狀態。統計數字**僅計入當前旅程內、且已蓋印的記錄**。

## 欄位說明

| 欄位 | 說明 |
|------|------|
| 學員編號 | 學員編號 |
| 姓名 | 學員姓名 |
| 法名 | 法名 |
| 班級 | 學員目前報名中（尚未完成）的班級名稱，多個班級以「/」分隔 |
| 八大修行項目 | 各項目顯示進度條與目前數量 / 目標數量（例如「15/365」） |
| 旅程起訖日 | 學員目前有效旅程的起始與結束日期（例如 `2024-01-01 ~ 2026-12-31`） |
| 達標 | 所有項目皆達目標門檻則顯示綠色「是」，否則顯示「否」 |

::: tip 統計範圍
所有修行項目的數字僅統計：
- 記錄日期落在學員**當前有效旅程**內
- 記錄已由法師**蓋印確認**

未蓋印或旅程外的記錄不計入。
:::

## 搜尋與篩選

篩選列分為兩層：

**第一列（搜尋範圍）**
- **學員搜尋**：支援學員編號、姓名、法名模糊搜尋
- **班級搜尋**：依班級名稱模糊搜尋（對應「班級」欄）
- CT 管理員另有「篩選精舍」下拉（見下方說明）

**第二列（達標條件與檢視切換）**
- **類別**：國內 / 海外（中）/ 海外（英），不同類別有不同達標門檻
- **層級**：基礎 / 進階
- **只顯示進行中旅程**：勾選後僅顯示目前有有效旅程的學員（旅程起訖日非空）
- **達標狀態**：三段切換 `全部 / 已達標 / 未達標`
- 右側：**匯出 Excel** 按鈕

篩選列下方會顯示結果摘要：`共 N 筆，其中 M 筆達標（X%）`，方便快速掌握目前檢視範圍。

## 匯出 Excel

點擊「匯出 Excel」按鈕，可將目前畫面顯示的資料（含所有篩選條件）匯出為 `.xlsx` 檔案。

檔名格式：`達標情況統計_基礎_2026-04-15.xlsx`

匯出內容包含所有欄位，其中發心出坡細分為三欄：精舍小時、回山次數、連續天數。

## CT 管理員

CT 管理員可額外透過「篩選精舍」下拉選單查看特定精舍的達標情況。下拉選單支援輸入精舍名稱快速篩選。
