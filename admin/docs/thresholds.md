# 達標門檻設定

::: warning CT 管理員專屬
此頁面功能僅限 CT 管理員使用。
:::

達標門檻設定頁面允許 CT 管理員調整各學員類別在八大修行項目上的基礎與進階達標目標值。設定結果會直接影響「達標情況統計」頁面的進度條與達標判斷。

## 學員類別分頁

設定頁面分為三個分頁：

| 分頁 | 對應學員類別 |
|------|-------------|
| 內地學員 | 國內學員 |
| 海外（中） | 海外中文學員 |
| 海外（英） | 海外英文學員 |

## 設定方式

每個分頁列出八大修行項目，每項提供兩個欄位可輸入：
- **基礎目標**：基礎層級的達標數量
- **進階目標**：進階層級的達標數量

修改完成後點擊右上角「儲存設定」按鈕即可生效。

::: warning 注意
門檻值設為 `0` 時，該項目在達標統計中將顯示「—」並視為不計入達標判斷。
:::
