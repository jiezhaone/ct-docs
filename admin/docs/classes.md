# 班級列表

班級管理功能用於建立與管理各種課程班級。

## 新增班級

1. 點擊「新增班級」按鈕
2. 依序選擇：
   - **語言**：中文班 / English
   - **時段**：根據語言自動篩選可用時段
   - **班級**：根據語言與時段自動篩選可用班級（含週數）
3. 選填：
   - **期別**：例如「2026春季」
   - **開始日期** / **結束日期**
   - **授課法師**（CT 管理員可指定）
4. 點擊確認建立

::: tip 提示
班級的週數由系統預設範本決定，例如「初級班 40 週」或「初級班 A 20 週」。
:::

## 編輯班級

點擊班級列表中的「編輯」按鈕，可修改班級的所有欄位。

## 封存班級

點擊「封存」按鈕可將班級設為封存狀態，封存後：
- 班級不再出現於預設的活躍班級列表中
- 已報名的學員在 App 中仍可查看該班級記錄

::: warning 注意
封存操作無法復原，請確認後再執行。
:::

## CT 管理員

CT 管理員可：
- 透過「篩選精舍」下拉選單查看各精舍的班級
- 建立班級時需先選擇精舍
- 指定授課法師
