# 法師管理

::: warning CT 管理員專屬
此頁面功能僅限 CT 管理員使用。
:::

進入「法師管理」頁面，可查看、編輯所有精舍法師的帳號與權限設定。跨精舍蓋章權限**僅在編輯現有法師時**可設定，新增法師時不會出現此區塊，儲存後再進入編輯即可設定。

## 如何設定跨精舍蓋章權限

### 操作步驟

1. **登入管理員帳號**
   使用管理員（角色：管理員）帳號登入後台 admin portal。

2. **前往法師管理頁面**
   左側選單點選「法師管理」。

3. **找到要授權的法師**
   在列表中搜尋法師姓名或登入帳號，找到後點選「編輯」。

4. **設定跨精舍蓋章權限**
   在編輯視窗下方，找到「跨精舍蓋章權限」區塊，勾選要授予的出坡類型：
   - 精舍出坡
   - 回山出坡
   - 連續護持

   > 例如：若該法師負責在回山時為各精舍學員蓋印，僅勾選「回山出坡」即可。

5. **儲存**
   點選「確定」，系統會同時儲存法師資料與蓋章權限。

### 授權後的效果

法師登入後進入「發心出坡記錄」頁面：

- 在「篩選出坡方式」選擇已被授權的項目（如「回山出坡」）後，即可看到所有精舍學員的記錄，而非只有自己精舍的學員。
- 表格會多顯示「精舍」欄位，方便識別學員所屬精舍。
- 可使用「篩選精舍」下拉選單，縮小檢視範圍。
- 蓋印（單筆或批次）功能同樣適用於跨精舍記錄。

### 移除權限

同樣進入「法師管理」→「編輯」，取消勾選對應項目後儲存即可。
