# 學員報名與完成

點擊班級列表中的展開按鈕（列左側箭頭），可管理該班級的學員。

## 搜尋已報名學員

展開班級後，可在工具列中的「搜尋已報名學員」輸入框以學員編號、姓名或法名即時篩選列表。列表每頁顯示 20 筆，超過時可翻頁瀏覽。

## 報名學員

1. 展開班級列
2. 在搜尋框中輸入學員編號、姓名或法名
3. 選擇一位或多位學員
4. 點擊「加入」按鈕

::: tip 提示
搜尋框支援學員編號、姓名、法名三種搜尋方式。
已報名的學員不會出現在可選清單中。
:::

## 批量匯入學員

若需一次報名大量學員，可使用「批量匯入」功能上傳 CSV 或 Excel 檔案。

### 準備檔案

- 檔案格式：`.csv`、`.xlsx`、`.xls`
- 第一欄為學員編號（`student_no`）
- 若第一行為標題列，系統會自動略過

點擊「下載範本」可取得格式正確的 Excel 範本。

### 操作步驟

1. 展開班級列
2. 點擊「批量匯入」按鈕
3. 點擊「選擇檔案」上傳準備好的 CSV 或 Excel
4. 系統解析後顯示預覽清單，摘要列出各狀態筆數，每筆記錄會標示狀態：
   - **可加入**（綠色）— 學員存在且尚未報名，將被加入
   - **已在班**（橘色）— 學員已報名此班級，略過
   - **找不到**（紅色）— 查無此學員編號，略過
5. 可使用上方篩選列（全部 / 可加入 / 已在班 / 找不到）快速聚焦問題記錄
6. 確認無誤後點擊「確認加入（N 位）」

::: tip 提示
系統只會加入狀態為「可加入」的學員，已在班與找不到的記錄會自動略過，無需手動篩選。
:::

## 設定完成日期

當學員完成課程後：

1. 勾選要設定的學員（可多選）
2. 點擊「設定完成日期」按鈕
3. 選擇完成日期
4. 點擊確認

完成的學員會在「完成日期」欄位顯示綠色標籤。

::: tip 提示
未勾選任何學員時，「設定完成日期」按鈕會停用；將滑鼠移至按鈕上會顯示提示「請先勾選下方學員」。
:::

## 批量匯入完成日期

若需一次設定多位學員的完成日期，可使用「匯入完成日期」按鈕上傳 CSV 或 Excel 檔案。

### 準備檔案

- 檔案格式：`.csv`、`.xlsx`、`.xls`
- 第一欄為學員編號（`student_no`）
- 第二欄為完成日期（`completed_at`，格式 `YYYY-MM-DD`，亦接受 `YYYY/MM/DD` 或 Excel 日期格式）
- 若第一行為標題列，系統會自動略過

點擊「下載範本」可取得格式正確的 Excel 範本。

### 操作步驟

1. 展開班級列，點擊「匯入完成日期」按鈕
2. 上傳檔案後，系統解析並逐筆比對現有資料，列出每筆狀態：
   - **可設定**（綠色）— 學員在班且尚未有完成日期
   - **將覆蓋**（橘色）— 學員已有完成日期，將以新日期覆蓋
   - **無變更**（藍色）— 完成日期與現有資料相同
   - **未在班**（紅色）— 學員未報名此班級
   - **找不到**（紅色）— 查無此學員編號
   - **日期錯誤**（紅色）— 完成日期格式無法解析
3. 可使用上方分段切換快速聚焦特定狀態
4. 若有「將覆蓋」記錄，可透過勾選框決定是否一併覆蓋
5. 確認後點擊「確認套用（N 筆）」

::: tip 提示
僅「可設定」與（勾選後的）「將覆蓋」記錄會被寫入，其餘狀態自動略過。
:::

## 設定為未完成

如需撤銷完成狀態：

1. 勾選要撤銷的學員
2. 點擊「設定為未完成」按鈕
3. 確認後完成日期會被清除

## 移除學員

點擊學員列右側的刪除按鈕，確認後將該學員從班級中移除。
