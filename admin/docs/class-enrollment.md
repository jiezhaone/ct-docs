# 學員報名與完成

點擊班級列表中的展開按鈕（列左側箭頭），可管理該班級的學員。

## 報名學員

1. 展開班級列
2. 在搜尋框中輸入學員編號、姓名或法名
3. 選擇一位或多位學員
4. 點擊「加入」按鈕

::: tip 提示
搜尋框支援學員編號、姓名、法名三種搜尋方式。
已報名的學員不會出現在可選清單中。
:::

## 設定完成日期

當學員完成課程後：

1. 勾選要設定的學員（可多選）
2. 點擊「設定完成日期」按鈕
3. 選擇完成日期
4. 點擊確認

完成的學員會在「完成日期」欄位顯示綠色標籤。

## 設定為未完成

如需撤銷完成狀態：

1. 勾選要撤銷的學員
2. 點擊「設定為未完成」按鈕
3. 確認後完成日期會被清除

## 移除學員

點擊學員列右側的刪除按鈕，確認後將該學員從班級中移除。
