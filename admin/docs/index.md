## 手冊內容

本手冊涵蓋福慧學習護照管理後台的所有功能操作說明，適用於精舍法師（DM）及 CT 管理員。

### 主要功能

- **學員管理** — 新增、編輯、匯入學員資料
- **班級管理** — 建立班級、報名學員、設定完成狀態
- **活動記錄** — 查看學員提交的修行記錄、蓋印確認
- **統計報表** — 達標情況統計、活動趨勢分析
- **通知管理** — 發送通知公告給學員（CT 管理員）

### CT 管理員功能

- **法師管理** — 查看與編輯法師帳號及跨精舍蓋章權限
- **學員參與** — 全平台學員活躍度與提交趨勢分析
- **蓋印進度** — 監控全平台待蓋印積壓與各法師蓋章效率
- **達標門檻設定** — 調整各學員類別的基礎與進階達標目標值
- **修行內容管理** — 維護 App 中各修行項目的教材連結
