## 手冊內容

本手冊涵蓋 CT Passport 管理後台的所有功能操作說明，適用於精舍法師（DM）及 CT 管理員。

### 主要功能

- **學員管理** — 新增、編輯、匯入學員資料
- **班級管理** — 建立班級、報名學員、設定完成狀態
- **活動記錄** — 查看學員提交的修行記錄、蓋印確認
- **統計報表** — 達標情況統計、活動趨勢分析
