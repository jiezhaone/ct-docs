# 修行內容管理

::: warning CT 管理員專屬
此頁面功能僅限 CT 管理員使用。
:::

修行內容管理頁面用於維護八大修行項目的教材連結與顯示設定，學員在 App 中點選修行項目時即可看到對應的內容連結。

## 篩選與搜尋

可透過頂部的「篩選活動類型」下拉選單，依修行項目類別快速過濾列表內容。

## 欄位說明

| 欄位 | 說明 |
|------|------|
| ID | 系統自動產生的唯一識別碼 |
| 活動類型 | 所屬的修行項目（如：課程出席、共修念佛等） |
| 中文名稱 | 內容項目的中文顯示名稱 |
| 英文名稱 | 內容項目的英文顯示名稱 |
| 中文連結 | 中文版教材的連結網址 |
| 英文連結 | 英文版教材的連結網址 |
| 排序 | 同一活動類型下的顯示順序（數字越小越前面） |
| 啟用狀態 | 是否在 App 中對學員顯示 |

## 新增內容

點擊右上角「新增」按鈕，填寫以下欄位後確認：
- 活動類型
- 中文名稱 / 英文名稱
- 中文連結 / 英文連結（選填）
- 排序
- 啟用狀態

## 編輯與刪除

- 點擊列表中的「編輯」按鈕可修改現有內容項目的所有欄位。
- 點擊「刪除」按鈕可移除該項目，刪除後學員 App 中將不再顯示。
