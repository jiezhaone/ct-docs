# 蓋印進度

::: warning CT 管理員專屬
此頁面功能僅限 CT 管理員使用。
:::

蓋印進度頁面顯示全平台待蓋印記錄的積壓狀況與各精舍法師的蓋印效率。

## 積壓概覽

| 指標 | 說明 |
|------|------|
| 待蓋章總數 | 目前全平台未蓋印的記錄總筆數 |
| 超過 7 天 | 提交後超過 7 天仍未蓋印的記錄筆數 |
| 超過 30 天 | 提交後超過 30 天仍未蓋印的記錄筆數 |
| 最舊未蓋章 | 最早一筆待蓋印記錄的提交日期 |

## 每週平均蓋章時效

長條圖顯示近幾週的平均蓋章時效（單位：小時），顏色表示狀態：
- **綠色**：正常（48 小時內）
- **橘色**：注意（7 天內）
- **紅色**：嚴重（超過 30 天）

## 待蓋章（依精舍）

列表顯示各精舍的待蓋印筆數、最久未蓋印天數，以及嚴重程度標籤（正常 / 注意 / 嚴重）。

## 法師蓋章活動（近 30 日）

列表顯示各法師近 30 天的蓋章數量與平均蓋章時效，可用於了解各精舍法師的蓋印回應速度。
