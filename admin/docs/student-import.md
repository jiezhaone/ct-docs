# 匯入學員

支援 CSV 及 Excel（.xlsx / .xls）格式批次匯入學員。

## 法師（DM）匯入

### 檔案格式

欄位順序如下（第一欄為學員編號）：

| 欄位 | 說明 | 必填 |
|------|------|------|
| student_no | 學員編號 | 是 |
| name | 姓名 | 是 |
| dharma_name | 法名 | 否 |

::: tip 提示
第一行若為標題列（包含「student_no」、「學員」等字樣），系統會自動略過。
:::

### 操作步驟

1. 進入「學員管理」頁面，點擊「匯入學員」
2. 點擊「選擇檔案」上傳 CSV 或 Excel，或點擊「下載範本」取得格式正確的範本
3. 系統顯示預覽表格，確認資料正確
4. 點擊「確認匯入（N 筆）」

### 匯入結果

系統會顯示：
- **新增**：成功匯入的筆數
- **略過**：因學員編號重複而略過的筆數

## CT 管理員匯入

::: warning CT 管理員專屬
CT 管理員可跨精舍匯入學員，需額外指定精舍名稱。
:::

### 檔案格式

欄位順序如下：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 精舍名稱 | 所屬精舍的名稱（需與系統一致） | 是 |
| student_no | 學員編號 | 是 |
| name | 姓名 | 是 |
| dharma_name | 法名 | 否 |

### 操作步驟

1. 進入「學員管理」頁面，點擊「匯入學員」
2. 點擊「選擇檔案」上傳 CSV 或 Excel，或點擊「下載範本」取得格式正確的範本
3. 系統顯示預覽表格，確認精舍名稱、學員編號等資料正確
4. 點擊「確認匯入（N 筆）」

### 匯入結果

系統會顯示：
- **新增**：成功匯入的筆數
- **略過**：因資料錯誤或學員編號重複而略過的筆數
- **略過明細**：列出每筆略過記錄的學員編號與原因
