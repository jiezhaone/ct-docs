# 通知管理

通知管理功能僅限 CT 管理員使用，可向學員發送通知公告。

## 新增通知

1. 點擊「新增通知」按鈕
2. 填寫以下欄位：
   - **中文標題** / **英文標題**（必填）
   - **中文內容** / **英文內容**（必填）
   - **指定精舍**：選擇特定精舍，或留空表示發送給所有精舍
   - **平台**：全部 / 網頁版 / Android / iOS
   - **排程發送**：可設定未來時間發送，留空則立即發送
3. 點擊確認建立

## 編輯通知

點擊通知列表中的「編輯」按鈕，可修改通知的所有欄位。

## 刪除通知

點擊「刪除」按鈕可刪除通知（軟刪除），已讀記錄將一併失效。

## 通知狀態

| 狀態 | 說明 |
|------|------|
| 已發佈 | 通知已即時發送，學員可立即收到 |
| 排程中 | 通知已建立但尚未到達排程時間 |

## 學員端行為

- 學員開啟 App 時會自動獲取未讀通知數量
- 點擊通知後自動標記為已讀
- 通知列表支援下拉重新整理與無限捲動分頁
