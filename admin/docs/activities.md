# 記錄管理與蓋印

學員透過 App 提交修行記錄後，法師需在管理後台進行蓋印確認。

## 八大修行項目

| 項目 | 說明 |
|------|------|
| 薰習課程記錄 | 課程出席記錄 |
| 誦持經典與共修記錄 | 經典誦讀與共修活動 |
| 誦讀法語記錄 | 開山祖師法語 |
| 研讀開示記錄 | 開山祖師開示（中級班以上） |
| 發願記錄 | 四弘誓願、普賢十大願王 |
| 反省記錄 | 身口意三業反省 |
| 發心出坡記錄 | 精舍出坡、回山出坡、連續護持 |
| 靜坐記錄 | 自行靜坐、共修靜坐 |

## 蓋印狀態切換

每個記錄頁面上方提供「**待蓋印 / 已蓋印 / 全部**」分段切換，預設顯示「待蓋印」。搜尋與其他篩選條件僅作用於當前選取的視圖。

- **待蓋印**：尚未蓋印的記錄，可進行蓋印操作
- **已蓋印**：已完成蓋印的記錄，僅供查詢
- **全部**：同時顯示兩種狀態

## 蓋印操作

### 單筆蓋印
點擊記錄右側的「蓋印」按鈕。

### 批次蓋印
1. 勾選多筆待蓋印的記錄
2. 點擊「批次蓋印」按鈕

::: tip 提示
已蓋印的記錄會顯示綠色「已蓋印」標籤，待蓋印的記錄顯示橘色「待蓋印」標籤。已蓋印的列不可勾選，因此批次蓋印僅對待蓋印的列生效。
:::

## 特殊欄位

### 發心出坡記錄
- **出坡方式**：精舍出坡 / 回山出坡 / 連續護持
- **數量**：依出坡方式顯示數量與單位（例如「2 次」、「3 天」）

#### 跨精舍蓋章（需具備授權）
若法師已被 CT 管理員授予跨精舍蓋章權限，在「篩選出坡方式」選擇已授權的項目後：
- 可看到所有精舍學員的記錄，而非只有自己精舍的學員
- 表格會多顯示「精舍」欄位，方便識別學員所屬精舍
- 可使用「篩選精舍」下拉選單縮小檢視範圍
- 單筆與批次蓋印功能同樣適用

### 靜坐記錄
- **靜坐方式**：自行靜坐 / 共修靜坐
- **數量**：支香數
