# 匯入學員

支援 CSV 及 Excel（.xlsx / .xls）格式批次匯入學員。

## 檔案格式

欄位順序如下：

| 欄位 | 說明 | 必填 |
|------|------|------|
| qrcode | 學員編號 | 是 |
| name | 姓名 | 是 |
| dharma_name | 法名 | 否 |

::: tip 提示
第一行若為標題列（包含「qrcode」字樣），系統會自動略過。
:::

## 操作步驟

1. 進入「學員管理」頁面，點擊「匯入檔案」
2. 選擇 CSV 或 Excel 檔案
3. 系統會顯示預覽表格，確認資料正確
4. 點擊「確認匯入」

## 匯入結果

系統會顯示：
- **新增**：成功匯入的筆數
- **略過**：因學員編號重複而略過的筆數
