# 個人資料

## 查看個人資料

點擊底部導覽列的「我的」即可查看個人資料，包含：
- 姓名 / 法名
- **已修行天數**：從最早一段旅程起始日至今的累計天數
- 學員編號（可點擊複製）
- 所屬精舍與區域
- 學員類別（國內 / 海外中 / 海外英）
- 時區設定

## 複製學員編號

學員編號旁有複製圖示，點擊即可將編號複製到剪貼簿，方便分享或填寫表單。

## 意見回饋

點擊「意見回饋」按鈕，可直接開啟電子郵件撰寫畫面，向管理團隊提供建議或回報問題。

## 登出

點擊「登出」按鈕即可登出目前帳號。

## 時區說明

時區由系統在**登入時自動偵測**，根據裝置目前的時間設定（UTC 偏移）記錄，格式如 `+08:00`。

- 使用者無法手動修改時區
- 若顯示「未設定」，重新登入一次即可自動更新
- 若您更換裝置或變更裝置時區後，重新登入即可同步最新的時區資訊

::: tip 提示
時區會影響修行記錄的日期歸屬。建議確認裝置時區設定正確，以確保記錄日期準確。
:::

## App 版本

個人資料頁面底部會顯示目前的 App 版本號，例如：

```
v1.2.0 (build 42)
```

部分版本會附帶內部識別碼，例如 `v1.2.0 (build 42 · abc1234)`。如需向法師或管理員回報問題，建議一併提供此版本資訊。

## 修改資料

目前個人資料暫時無法自行修改，如需更新請聯繫您所屬精舍的法師。
