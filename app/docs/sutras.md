# 修習內容管理

## 查看經典列表

在「誦持經典與共修」項目中，您可以通過點擊「誦讀」或詳情頁中的「管理經典」查看系統預設的經典列表。用戶需要從中選擇正在修習的經典。

## 提交經典記錄

1. 點擊「誦持經典與共修」中的「記錄」
2. 選擇要記錄的經典
3. 輸入數量
4. 點擊「提交」

## 誦讀法語圖片瀏覽

在「誦讀法語」項目中，點擊「誦讀」可進入法語圖片瀏覽器：

- **左右滑動**：切換上一則 / 下一則法語
- **雙指縮放**：放大圖片查看細節（最大 4 倍），縮放時左右滑動暫時停用
- **書籤功能**：可收藏常讀的法語，方便快速查找
- **自動記憶**：下次開啟時會自動跳至上次閱讀的位置

### 網頁版額外功能
- 左右箭頭按鈕可切換法語
- 鍵盤方向鍵（← →）可快速翻頁

## 研讀開示

在「研讀開示」項目中，點擊「研讀」可開啟開示內容。頁面中若有連結跳轉，返回鍵會優先在頁面內返回上一層，而非直接關閉頁面。

### 管理開示列表

與經典列表管理類似，用戶可以從系統預設的開示列表中選擇正在研讀的開示：

1. 進入「研讀開示」詳情頁
2. 點擊「管理開示」
3. 勾選或取消勾選想要納入個人列表的開示
4. 選擇結果會自動儲存

提交研讀記錄時，僅需輸入數量與日期即可，無需逐筆指定開示項目。
