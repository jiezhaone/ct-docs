# 登入與註冊

## 登入步驟

1. 開啟福慧學習護照 App
2. 輸入您的學員編號
3. 點擊「驗證」
4. 確認顯示的個人資訊是否正確
5. 點擊「確認登入」

::: tip 提示
學員帳號由精舍法師建立。如尚未擁有帳號，請聯繫您的精舍法師。
:::

## 快速登入

App 會自動記錄您最近成功登入的學員編號（最多 5 組）。下次登入時，輸入框下方會顯示「最近登入」列表，點擊即可快速填入編號，無需重新輸入。

::: tip 提示
快速登入記錄儲存在裝置本機，更換裝置或清除 App 資料後需重新輸入。
:::

## 網頁版

福慧學習護照也提供網頁版，可透過瀏覽器訪問 [app.ct-passport.org](https://app.ct-passport.org)，操作方式與 App 相同。
