# 報數（提交記錄）

## 如何報數

1. 在首頁點擊您要記錄的修行項目
2. 輸入當日的修行數量
3. 若該項目有子類別（如出坡方式、靜坐方式），請先選擇類別
4. 點擊「提交」完成報數

## 報數規則

- 每個項目每日可提交一次或多次
- 提交後可在「歷史記錄」中查看
- 記錄需經法師蓋印確認後才計入護照進度

::: warning 注意
已提交的記錄可以刪除，但已蓋印的記錄無法刪除。
:::

## 修改記錄

目前僅支援修改記錄的**數量**，日期與項目類別無法變更。

1. 在首頁點擊要管理的修行項目，進入該項目的詳細頁面
2. 在本月記錄中找到要修改的那筆，點擊數量文字（旁邊有鉛筆圖示）
3. 底部彈出修改面板，使用「＋」「－」按鈕調整數量
4. 點擊「確認」儲存

::: warning 注意
已蓋印的記錄無法修改，數量文字不可點擊。
:::

## 刪除記錄

1. 在首頁點擊要管理的修行項目，進入該項目的詳細頁面
2. 頁面下方顯示本月的報數記錄，提示文字顯示「左滑刪除」
3. 在要刪除的記錄上**向左滑動**，右側出現紅色刪除圖示
4. 點擊刪除圖示，確認對話框後點擊「刪除」完成

::: warning 注意
- 已蓋印（顯示綠色「已蓋章」標籤）的記錄無法滑動刪除
- 刪除後無法復原，請確認後再操作
:::

## 補報

如果忘記報數，可以補報過去日期的記錄：

1. 在報數頁面點擊日期欄位
2. 選擇需要補報的日期
3. 輸入該日的修行數量
4. 點擊「提交」完成補報

## 發心出坡特別說明

在記錄發心出坡時，切換出坡方式後，數量會自動重設為該方式的預設值：
- 精舍出坡：預設 1 小時
- 回山出坡：預設 1 次
- 連續護持：預設 3 天
