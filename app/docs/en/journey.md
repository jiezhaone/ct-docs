# Achievements & Progress

## Starting a New Journey

A journey (one year) is the basic cycle for passport records — all logged practice and goal calculations are scoped to your current journey.

If you do not have an active journey, the "Achievements" page will show a "Start a New Journey" card:

1. Tap "Set Start Date"
2. Pick a start date (must be between January 1 of the current year and today)
3. Confirm to create a one-year journey

::: tip
Each student can have only one active journey at a time. When a journey ends, you can start a new one from the "Achievements" page.
:::

## Current Journey

Shows your current passport level and progress for each practice activity. Each activity is shown as a progress bar, so you can see how much more is needed to reach the goal.

## Past Journeys

Records your previously completed passport journeys, including completion dates and final results for each activity.
