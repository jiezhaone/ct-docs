## Navigation Overview

The app has four main tabs at the bottom:

| Tab | Function |
|------|------|
| **Home** | Log practice, see today's progress; tap the bell icon (top right) to open notifications |
| **History** | Browse past practice records in a calendar view |
| **Achievements** | Start a new journey, track passport progress and past journeys |
| **Profile** | Personal info, language switch, feedback, and logout |

## Manual Contents

This manual covers all features of the CT Passport mobile app and is intended for all practitioners.

### Main Features

- **Start a Journey** — Choose a start date to begin a one-year practice cycle.
- **Log Practice** — Submit daily records across the eight practice activities; you can edit or delete records that have not yet been stamped.
- **History** — Review your daily records in a calendar view.
- **Manage Practice Content** — Curate your personal lists of sutras and Dharma lectures.
- **My Courses** — View the classes you are enrolled in.
- **Achievements & Progress** — Track passport progress, current journey, and journey history.
- **Notifications** — Receive announcements from your center and the system.
- **Profile** — Manage personal info, language preference, and feedback.
