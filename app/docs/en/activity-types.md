# The Eight Practice Activities

CT Passport tracks the following eight practice activities:

| Activity | Description | Unit |
|------|------|------|
| Course Attendance | Completed classes, converted by week count (20 weeks = 1 entry) | class |
| Sutra Recitation & Group Practice | Sutra recitation and group practice events | volume |
| Dharma Excerpts | Excerpts of the Founding Master's Dharma words | excerpt |
| Dharma Lectures | The Founding Master's lectures (intermediate level and above) | lecture |
| Vow Making | Four Great Vows, Ten Great Vows of Samantabhadra | time |
| Self Reflection | Reflection on the three karmas (body, speech, mind) | time |
| Meritorious Service | Center service / Mountain-retreat service / Continuous service | hour / time / day |
| Sitting Meditation | Self meditation / Group meditation | stick of incense |

## Meritorious Service Subcategories

Meritorious Service has three service types with different units:

| Service Type | Unit | Description |
|----------|------|------|
| Center Service | hour | Counted in actual hours of service (1 hour = 1 hour) |
| Mountain-retreat Service | time | Mountain service of less than 2 days; each session must be at least 3 hours |
| Continuous Service | day | Mountain service of 3 days / 2 nights or more (minimum 3 days) |

::: tip
Goal thresholds vary by passport level — see the "Passport Progress" page for details.
:::

::: warning Course Attendance — How It's Counted
"Course Attendance" is **not** counted per class session, but by **completing a 20-week standard class** as 1 entry:

- The class must be **completed** (drops or in-progress classes do not count)
- The completion date must fall within the active journey
- Class length is converted by week count:
  - A 20-week class = **1 entry**
  - A 10-week class = **0.5 entry**
  - A 40-week class = **2 entries**
:::
