# History

The History screen offers a calendar view for browsing and querying past practice records.

## Opening History

Tap "History" in the bottom navigation bar.

## Calendar View

The current month's calendar is shown at the top:
- **Red circle**: practice was logged on that day
- **Dark circle**: currently selected date
- **Today**: outlined in gold
- A light-red band connects consecutive days with records

Tap any date to update the list below. Use the left/right arrows (or swipe the calendar) to switch months — only up to the current month is browsable.

## Record List

After selecting a date, all records for that day are shown in chronological order. Each entry shows:
- Practice activity name and subcategory (e.g., recitation type, service type)
- Submission time
- Quantity and unit
- Whether it has been stamped (green "Stamped" tag)

The list supports pull-to-refresh, and more records load automatically when you scroll to the bottom.

::: tip
History is read-only — records cannot be edited or deleted here. To make changes, open the corresponding activity's detail page from the Home screen.
:::
