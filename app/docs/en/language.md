# Language Switch

## Supported Languages

CT Passport supports:
- Traditional Chinese
- English

## How to Switch

### From the Login Screen
Tap "EN" or "中" in the top-right corner.

### Inside the App
Tap "Home" in the bottom navigation bar, then tap "中" or "EN" in the top-right corner.

::: tip
After switching, all UI text updates immediately. On the web version, the browser tab title also switches to the matching language.
:::
