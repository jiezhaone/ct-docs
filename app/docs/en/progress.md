# Passport Progress

## Viewing Progress

Tap a practice activity on the Home screen to open its detail page, which shows:
- **Today / Yesterday**: counts logged today and yesterday
- **Total**: cumulative count within the current journey
- **Days**: number of days within the current journey on which you logged practice
- **Practice Goals**: progress toward Basic and Advanced thresholds

## Practice Goals

Each activity has two goal levels — **Basic** and **Advanced** — shown as progress bars. Goal thresholds vary by student type (Taiwan / Overseas-CN / Overseas-EN).

### Meritorious Service — Special Goal Criteria

Meritorious Service uses a composite goal rule, with multiple ways to qualify:

**Basic** (meet any one):
- Path 1 (service hours): 60 hours of center service; or 1 mountain-retreat service + 45 hours of center service
- Path 2 (continuous service): 3 consecutive days of continuous service

**Advanced** (meet any one):
- Path 1: meet the Basic service-hours condition + 3 consecutive days of continuous service
- Path 2: 8 cumulative days of continuous service
- Path 3: 120 cumulative hours of center service

::: tip
The Meritorious Service goal section is collapsed by default and only shows Basic / Advanced status. Tap to expand and see the detailed progress of each path.
:::

## Calendar & Records

The activity detail page also includes:
- **Monthly Calendar**: shows daily counts; stamped dates are highlighted
- **Log List**: all records for the month; unstamped records can be deleted

## Passport Levels

When all activities reach the goal threshold, you advance to the next passport level.
