# Profile

## Viewing Your Profile

Tap "Profile" in the bottom navigation bar to view your personal information, including:
- Name / Dharma name
- **Days practicing**: cumulative days since the start of your earliest journey
- Student number (tap to copy)
- Center and region
- Student type (Taiwan / Overseas-CN / Overseas-EN)
- Timezone

## Copying Your Student Number

A copy icon appears next to the student number — tap it to copy the number to the clipboard for easy sharing or form filling.

## User Manual

Tap the "User Manual" button to open this document inside the app and reference any feature at any time.

## Feedback

Tap the "Feedback" button to open the email composer and send suggestions or report issues to the management team.

## Logout

Tap the "Logout" button to sign out of the current account.

## About the Timezone

The timezone is **auto-detected at login** from your device's current time setting (UTC offset), in the format `+08:00`.

- Users cannot edit the timezone manually
- If it shows "Not set", logging in again will refresh it
- After switching device or changing device timezone, log in again to sync the new value

::: tip
The timezone affects which day a record belongs to. Make sure your device's timezone is set correctly so records land on the right date.
:::

## App Version

The bottom of the Profile page shows the current app version, e.g.:

```
v1.2.0 (build 42)
```

Some builds include an internal identifier, e.g., `v1.2.0 (build 42 · abc1234)`. When reporting issues to a master or admin, please include this version info.

## Editing Your Info

Personal information is not currently editable. To update it, please contact a master at your center.
