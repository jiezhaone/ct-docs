# My Courses

## My Courses

On the "Course Attendance" activity page, you can see the classes you are enrolled in.

Courses are shown in two sections:

### In Progress

Currently active classes, with:
- Class name (time slot + class name)
- Number of weeks
- Term
- Start / end dates

### Completed

Completed classes, with:
- Class name
- Completion date (highlighted in green)

## Practice Goal

The Course Attendance goal is calculated by **completed classes**, not by attendance counts, and is converted by the class's **week count** (20 weeks = 1 entry):

- Completing one 20-week class = **1 entry**
- Completing one 10-week class = **0.5 entry**
- Completing one 40-week class = **2 entries**

Drops and in-progress classes do not count.

## Attendance Records

For Course Attendance, the records section is labeled "Attendance Records" — each record represents one class check-in.

::: tip
Classes are created and enrolled by the center's master — students cannot self-enroll. To register, please contact your master.
:::
