# 查看課程

## 我的課程

在「薰習課程」項目頁面中，您可以查看已報名的班級資訊。

課程分為兩個區塊：

### 進行中

顯示目前正在進行的課程，包含：
- 課程名稱（時段 + 班級名稱）
- 課程週數
- 學期
- 起訖日期

### 已完成

顯示已完成的課程，包含：
- 課程名稱
- 完成日期（綠色標示）

## 修行目標

薰習課程的修行目標以**已圓滿完成的課程期數**計算，而非簽到次數。例如：
- 已圓滿完成課程 2 期
- 還未完成課程

## 簽到記錄

薰習課程的記錄標題顯示為「簽到記錄」，每筆記錄代表一次課程簽到。

::: tip 提示
課程由精舍法師建立並報名，學員無法自行加入課程。如需報名，請聯繫您的精舍法師。
:::
