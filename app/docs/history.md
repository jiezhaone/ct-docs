# 歷史記錄

歷史記錄頁面提供日曆視圖，讓您瀏覽與查詢過去的修行記錄。

## 開啟歷史記錄

點擊底部導覽列的「歷史」即可進入。

## 日曆視圖

頁面上方顯示當月日曆：
- **紅色圓圈**：該日有提交修行記錄
- **深色圓圈**：目前選取的日期
- **今日**：以金色外框標示
- 有連續記錄的日期之間會顯示淡紅色連接色帶

點擊任一日期，下方列表即更新為該日的記錄。點擊左右箭頭（或左右滑動日曆）可切換月份，最多只能瀏覽到當前月份。

## 記錄列表

選取日期後，下方依時間顯示當日的所有修行記錄，每筆顯示：
- 修行項目名稱與子類別（如誦持方式、出坡方式）
- 提交時間
- 數量與單位
- 是否已蓋章（綠色「已蓋章」標籤）

列表支援下拉刷新，滑到底部會自動載入更多記錄。

::: tip 提示
歷史記錄僅供查閱，無法在此頁面直接修改或刪除記錄。如需修改，請至首頁進入對應的修行項目詳細頁面操作。
:::
