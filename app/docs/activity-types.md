# 八大修行項目

CT Passport 追蹤以下八大修行項目：

| 項目 | 說明 | 單位 |
|------|------|------|
| 薰習課程 | 參加精舍開設的課程 | 堂 |
| 讀誦經典 | 誦讀佛經 | 部 |
| 抄寫經典 | 抄寫佛經段落 | 則 |
| 聽經聞法 | 聆聽佛法開示 | 小時 |
| 發願迴向 | 發願與迴向功德 | 次 |
| 懺悔反省 | 自我反省與懺悔 | 次 |
| 發心出坡 | 義工服務（出坡方式：可選擇具體類別） | 天/次 |
| 靜坐 | 禪修靜坐（可選擇自行靜坐或共修靜坐） | 支香 |

::: tip 提示
各項目的達標數量依護照等級而異，詳情請參考「護照進度」頁面。
:::
