# 八大修行項目

福慧學習護照追蹤以下八大修行項目：

| 項目 | 說明 | 單位 |
|------|------|------|
| 薰習課程 | 已結業的課程，依週數換算（20 週 = 1 筆） | 班 |
| 誦持經典與共修 | 經典誦讀與共修活動 | 部 |
| 誦讀法語 | 開山祖師法語 | 則 |
| 研讀開示 | 開山祖師開示（中級班以上） | 篇 |
| 發願 | 四弘誓願、普賢十大願王 | 次 |
| 反省 | 身口意三業反省 | 次 |
| 發心出坡 | 精舍出坡 / 回山出坡 / 連續護持 | 小時 / 次 / 天 |
| 靜坐 | 自行靜坐 / 共修靜坐 | 支香 |

## 發心出坡子類別

發心出坡包含三種出坡方式，各有不同的計量單位：

| 出坡方式 | 單位 | 說明 |
|----------|------|------|
| 精舍出坡 | 小時 | 以實際服務時數計算（1 小時 = 1 小時） |
| 回山出坡 | 次 | 回山出坡 2 天以下，每次至少 3 小時以上 |
| 連續護持 | 天 | 回山出坡 3 天 2 夜以上（最少 3 天） |

::: tip 提示
各項目的達標數量依護照等級而異，詳情請參考「護照進度」頁面。
:::

::: warning 薰習課程的計算方式
「薰習課程」**不是以上一次課為 1 筆**，而是以**完成一個 20 週標準班**為 1 筆：

- 必須**結業**才會列入（中途退選或仍在進行中的班都不計）
- 結業日期必須落在當前有效旅程內
- 班級長度依週數換算：
  - 20 週的班 = **1 筆**
  - 10 週的班 = **0.5 筆**
  - 40 週的班 = **2 筆**
:::
