# 八大修行項目

福慧學習護照追蹤以下八大修行項目：

| 項目 | 說明 | 單位 |
|------|------|------|
| 薰習課程 | 課程出席記錄 | 堂 |
| 誦持經典與共修 | 經典誦讀與共修活動 | 部 |
| 誦讀法語 | 開山祖師法語 | 則 |
| 研讀開示 | 開山祖師開示（中級班以上） | 篇 |
| 發願 | 四弘誓願、普賢十大願王 | 次 |
| 反省 | 身口意三業反省 | 次 |
| 發心出坡 | 精舍出坡 / 回山出坡 / 連續護持 | 天/次 |
| 靜坐 | 自行靜坐 / 共修靜坐 | 支香 |

::: tip 提示
各項目的達標數量依護照等級而異，詳情請參考「護照進度」頁面。
:::
