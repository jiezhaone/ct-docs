# 成就與進度

## 開啟新旅程

旅程（為期一年）是護照記錄的基本週期，所有報數、達標統計都會以當前旅程為範圍計算。

若您目前沒有進行中的旅程，「成就」頁面會顯示「開啟新旅程」卡片：

1. 點擊「設定起始日期」
2. 選擇旅程起始日（須為當年度的 1 月 1 日至今天之間）
3. 確認後即會建立一段為期一年的旅程

::: tip 提示
每位學員同一時間僅能有一段進行中的旅程。旅程結束後，可再次於「成就」頁面開啟新的旅程。
:::

## 當前旅程

顯示您目前所在的護照等級及各修行項目的完成進度。每個項目以進度條呈現，方便您掌握距離達標還需多少數量。

## 修行歷程

記錄您過去已完成的護照旅程，包含每一階段的完成時間與各項目的最終成績。
