## 導覽說明

App 底部有四個主要頁籤：

| 頁籤 | 功能 |
|------|------|
| **首頁** | 報數、查看今日進度；右上角鈴鐺可開啟通知 |
| **歷史** | 以日曆視圖瀏覽過去的修行記錄 |
| **成就** | 開啟新旅程、追蹤護照進度與修行歷程 |
| **我的** | 個人資料、語言切換、意見回饋與登出 |

## 手冊內容

本手冊涵蓋福慧學習護照行動應用程式的所有功能操作說明，適用於所有修行學員。

### 主要功能

- **開啟旅程** — 自行設定起始日期，建立為期一年的修行記錄週期
- **修行記錄** — 每日報數，記錄八大修行項目；支援修改與刪除未蓋印記錄
- **歷史記錄** — 以日曆視圖查詢過去每日的修行記錄
- **修習內容管理** — 自選誦持經典與研讀開示的項目清單
- **我的課程** — 查看已報名的薰習課程
- **成就與進度** — 追蹤護照進度、當前旅程與修行歷程
- **通知訊息** — 接收精舍與系統的通知公告
- **個人設定** — 管理個人資料、語言偏好與意見回饋
