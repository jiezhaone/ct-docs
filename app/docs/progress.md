# 護照進度

## 查看進度

點擊首頁的修行項目，進入項目詳情頁即可查看：
- **今日 / 昨日**：當日與昨日的報數數量
- **總計**：當前旅程內的累計數量
- **天數**：當前旅程內有報數的天數
- **修行目標**：基礎與進階的達標進度

## 修行目標

每個修行項目設有「基礎」與「進階」兩個等級的達標目標，以進度條呈現。達標數量依學員類別（台灣 / 海外(中) / 海外(英)）不同而異。

### 發心出坡的特殊達標標準

發心出坡的達標標準為複合式條件，包含多種達標方式：

**基礎**（符合其一即達標）：
- 方式一（發心出坡）：精舍出坡累計 60 小時；或回山出坡 1 次 + 精舍出坡累計 45 小時
- 方式二（連續護持）：連續護持滿 3 天

**進階**（符合其一即達標）：
- 方式一：同時完成基礎的發心出坡條件 + 連續護持滿 3 天
- 方式二：連續護持累積滿 8 天
- 方式三：精舍出坡累計滿 120 小時

::: tip 提示
發心出坡的修行目標預設為收合狀態，僅顯示基礎與進階的達標狀態。點擊可展開查看各方式的詳細進度。
:::

## 日曆與記錄

項目詳情頁還包含：
- **月曆**：顯示每日的報數數量，已蓋印的日期會特別標示
- **報數記錄**：該月的所有報數記錄列表，可刪除未蓋印的記錄

## 護照等級

完成所有項目的達標數量後，即可晉升至下一護照等級。
