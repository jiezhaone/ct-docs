# CT Passport — Documentation

User manuals for the CT Passport (福慧學習護照) platform, built with [VitePress](https://vitepress.dev/).

## Live Documentation

- **App Manual (學員手冊):** https://ct-passport.org/app-manual/
- **Admin Manual (管理後台手冊):** https://ct-passport.org/admin-manual/

## Local Development

```bash
npm install

# App manual
npm run dev:app

# Admin manual
npm run dev:admin
```

## Deployment

Push to `main` triggers automatic deployment to production via GitHub Actions.
